This repo teaches AI Agents how to THINK.

Agent Skills for Context Engineering: the open-source skill library that every Claude Code and Cursor user needs.

Here's why context engineering > prompt engineering:

Prompt engineering: "how do I phrase this better?"
Context engineering: "what information does the AI need, and how do I structure it?"

One improves a single message. The other improves EVERY message.

This repo has 14 skills across 5 categories:

🔹 Foundations
1️⃣ Context Fundamentals: what context is, why it matters, anatomy of agent systems
2️⃣ Context Degradation: lost-in-middle, poisoning, distraction, clash
3️⃣ Context Compression: strategies for long-running sessions
4️⃣ Context Optimization: compaction, masking, caching

🔹 Architecture
5️⃣ Multi-Agent Patterns: orchestrator, peer-to-peer, hierarchical
