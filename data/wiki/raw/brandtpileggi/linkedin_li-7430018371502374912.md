I finally found the Agent Interface for the AI swarm. It’s not an IDE or TUI. At these speeds, there is no User Interface for humans.

My setup:
- Headless Claude Code Agent Swarm
- MiniMax-M2.5 on vLLM 0.16.0rc2
- 4x A6000 (gen1, 2020)

The performance is bananas
- Prefill max (prompt ingest): 134,614 input tokens/sec @16 agents
- Decode max (generation): 1031 output tokens/sec @12 agents

My Human Interface is the Radicle distributed Git issues queue.
That’s where I see what’s done, what’s blocked, and where the swarm needs a human to break the tie. The queue is my new HUD: throughput in closed issues, latency in comment threads, errors as “needs repro”.

And yes, I still have trouble letting go and not watching… so I built a workflow where the only thing I “watch” is the backlog.

Make sure you own your AI. AI in the cloud is not aligned with you; it’s aligned with the company that owns it.
