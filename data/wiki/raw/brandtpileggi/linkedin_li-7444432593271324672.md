The biggest problem in AI coding right now is sycophancy.

Not in the broad "AI agrees with everything you say" sense, though that's real. Specifically this: when you have a coding agent evaluate its own work, it's a student grading its own homework. It'll flag a couple of minor issues to make the review look credible, but the deeper problems get stuffed under the rug.

The model was trained to agree. That bias doesn't disappear just because you asked it to be critical!

So the fix is architecture, not "prompting harder".

I built what I call an adversarial dev harness, inspired by how Anthropic is thinking about long-running coding agents. The core idea comes from GANs (generative adversarial networks), where a generator and a discriminator are trained against each other. Here, the generator writes code. The evaluator's sole job is to find flaws, not approve.

The architecture looks like this:

User prompt goes to a Planner, which expands it into a full spec. Then you enter a sprint cycle. Before a single line of code is written, the Generator and Evaluator negotiate the evaluation criteria together. They agree up front on what each sprint looks like and how it gets scored on a 1-10 scale. Only then does the Generator implement, and the Evaluator rips it apart against those pre-negotiated criteria.

Two core principles at work here:

1. Separate evaluation from coding entirely (different agents, isolated contexts, no shared bias)
2. Scope and contract before implementation (criteria set before code is written, not after)

The results are legitimately incredible. I used this harness to one-shot a full-stack RAG app with a polished chat UI, token streaming, source citations, the whole thing. Ran for about four hours. And I used Claude Sonnet 4.6 for it, not Opus (huge difference in capability)!

That's the real leverage: a better harness lets you use a cheaper, faster model and still get better output than a stronger model running alone.

I posted a full breakdown on YouTube (link to my channel in my bio) with code walkthroughs for both the Claude Code and Codex versions of the harness, plus a live demo of the RAG app it built.
