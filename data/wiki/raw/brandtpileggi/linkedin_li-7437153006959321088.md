LangChain just open-sourced their answer to Claude Code.

9.9K GitHub stars. 5 hours old.

Deep Agents: the batteries-included agent harness. MIT licensed. FREE.

Here's why this is a big deal:

Claude Code is amazing but closed-source and locked to Anthropic.

Deep Agents gives you the same architecture. Any model. Any provider.

What's included out of the box:

1️⃣ Planning
write_todos for task breakdown and progress tracking. The agent plans before it codes.

2️⃣ Filesystem access
read_file, write_file, edit_file, ls, glob, grep. Full codebase navigation.

3️⃣ Shell execution
Run commands with sandboxing. Build, test, deploy from the agent.

4️⃣ Sub-agents
Spawn isolated workers with their own context windows. Parallel work without context pollution.

5️⃣ Context management
Auto-summarization when conversations get long. Large outputs saved to files. The problem Claude Code solved, now OPEN SOURCE.

Two lines to start:

pip install deepagents

agent = create_deep_agent()

Works with Claude, OpenAI, Google, or any LangChain-compatible model. Swap providers without rebuilding.

Built on LangGraph. Streaming, persistence, checkpointing: all production-ready.

The CLI adds web search, remote sandboxes, persistent memory, custom skills, and human-in-the-loop approval.

From their README: "This project was primarily inspired by Claude Code."

They're not hiding it. They're DEMOCRATIZING it.

Are you still locked into a single AI coding provider? 👇
